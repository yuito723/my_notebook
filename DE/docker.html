    Docker

  • 詳しくはこちら

    • 使い方

    イメージのダウンロード

  • docker pull {imagename}{:tag}

    • コンテナの作成と起動

  • docker run --rm -it --name {containerID} -h {hostname} -p {sshIP}:22 {imagename}{:tag} {command(/bin/bash)}

    • 動作中のコンテナを表示

  • docker ps
  • docker ps -a(動作終了も含めて表示)

    • コンテナを削除

  • docker rm {containerID}

    • イメージを削除

  • docker rmi {imagename}

    • コンテナから抜ける

  • ctrl+P or ctrl+Q

    • コンテナに再接続する

  • docker attach {containerID or containername}

    • コンテナから抜けて終了する

  • exit

    • コンテナを再開

  • docker start {containerID or containername}

    • コンテナを終了する

  • docker stop {containerID or containername}

    • rosyuku/ubuntu-rdpコンテナを使用する場合

  • docker run --rm -it --name {containername} -h {hostname} -p {remotedesktopIP}:3389 -p {sshIP}:22 --shm-size=256m rosyuku/ubuntu-rdp{:tag}
